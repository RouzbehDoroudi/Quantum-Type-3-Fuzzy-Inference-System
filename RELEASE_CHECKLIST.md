# Release checklist

Before making the repository public:

- [ ] Confirm MIT License is acceptable.
- [ ] Replace `YOUR-USERNAME` in `CITATION.cff` after GitHub repository creation.
- [ ] Run all four scripts in fast mode.
- [ ] Optionally run selected scripts with `--use-qiskit`.
- [ ] Confirm no private local paths remain.
- [ ] Confirm no publisher PDF is included.
- [ ] Confirm external data redistribution rights before adding any data files.
- [ ] Create GitHub release `v1.0.0`.
- [ ] Archive the release on Zenodo and add the Zenodo DOI to README.

- [ ] Confirm shot-budget wording: 128 is only a smoke-test setting; 4096 or full sweep is recommended for paper-level runs.
